这是一个Java后台服务，主要用于处理用户请求和数据处理，使用SpringBoot框架。
所有Chat和Compose的交互都使用中文。
项目要求：
1. 使用SpringBoot框架
2. 使用MyBatis框架
3. 使用SQLite数据库
4. 使用Redis缓存
5. 使用RabbitMQ消息队列
6. 使用docker部署
7. 使用Nginx做反向代理
8. 使用lombok简化代码
11. 使用Swagger生成API文档
12. 使用Maven做依赖管理
13. 修改了代码可以热更新
其他：
1. 项目结构清晰，代码规范，注释清晰
2. 自动生成开发文档，放在docs目录下
